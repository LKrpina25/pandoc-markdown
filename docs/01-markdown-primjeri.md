# Markdown
## Osnovna sintaksa
Koristimo # * i **
### ako želim koristiti naslovne razine potrebno je unijeti sljedeću sintaksu
```bash
#  Naslov 1
##  Naslov 2
###  Naslov 3
####  Naslov 4
```
### ako želim koristiti kurziv potrebno je unijeti
```bash
*kurziv*
```
primjer *kurziv*

### ako želim koristiti podebljanje teksta potrebno je unijeti 
```bash
**Podebljanje**
```
primjer **Podebljanje**
ili 
```bash
__Podebljanje__
```
